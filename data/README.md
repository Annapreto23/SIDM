# Data Directory

This directory contains the input data files (HDF5 format) required for the analysis.

## Files
- `Halo_data_*.hdf5`: Simulation data containing halo properties and profiles.
- `dummy_halo.hdf5`: A small dummy dataset generated for testing purposes.

## Usage
Place your large HDF5 simulation files here. The analysis scripts are configured to look for data in this directory or accept paths to files residing here.
